VQAonline Dataset

Overview

VQAonline is the first to originate from an authentic VQA use case end to end, including with authentic context and answers, as well as the first VQA dataset sourced from an online question answering community.

VQAonline Dataset

The VQAonline dataset includes:

  • Images
    • train, validation, test
  • Annotations
    • Visual Question Answering. [JSON files]

Training, Validation, and Test Sets

  • Training Set
    • 665 examples
  • Validation Set
    • 285 examples
  • Test Set
    • 63,746 examples

Details about each visual question are in the following format:

"image:{"VQAonline_00000001.png",
"question":xxxx, 
"context":xxxxx,
"answers":xxxxx}"
